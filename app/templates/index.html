Cel aplikacji

Aplikacja służy do automatycznego pobierania opinii o produktach z serwisu Ceneo.pl na podstawie ich kodów identyfikacyjnych oraz do analizy i wizualizacji zebranych danych opinii w formie statystyk i wykresów.

Opis działania aplikacji

Użytkownik wprowadza kod produktu z Ceneo.pl, a aplikacja automatycznie pobiera wszystkie dostępne opinie wraz z szczegółowymi informacjami (autor, ocena gwiazdkowa, rekomendacja, treść, wady, zalety, daty itp.). Zebrane dane są następnie prezentowane w postaci interaktywnej tabeli z możliwością sortowania i filtrowania oraz eksportowane do różnych formatów plików

Zastosowane biblioteki

  • Flask - micro framework do tworzenia aplikacji webowych
  • Bootstrap 5 - framework CSS do responsywnego interfejsu użytkownika
  • DataTables - biblioteka JavaScript do zaawansowanych tabel interaktywnych
  • jQuery - biblioteka JavaScript do manipulacji DOM
  • BeautifulSoup/requests - biblioteki Python do web scrapingu (pobierania danych z Ceneo.pl)
  • pandas/csv/json - biblioteki do przetwarzania i eksportu danych

Zastosowane podejścia

  • Strukturalne
  • Obiektowe
